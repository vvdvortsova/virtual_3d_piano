=====Использование клавиатуры=====

++++++С помощью клавиатуры++++++

Чтобы пользоваться клавиатурой нужно
нажимать на клавиши клавиатуры вашей машины.
CAPS LOCK - самая нижняя белая клавиша клавиатуры и самая нижняя нота.
N - клавиша отвечает за ноту ДО второй октавы.
PAGE DOWN - последняя клавиша 2й октавы.
ENTER - не является клавишей для воспроизведения ноты.
Обратите внимание! SHIFT - нота РЕ первой октавы, второй SHIFT - тоже отвечает за эту ноту.
Черные ноты первой октавы начинаются c клавиши A. AS FGH
Черные ноты втророй октавы начинаются c клавиши J. JK L:
Обратите внимание! ENTER - не является клавишей для воспроизведения ноты.
Обратите внимание!TAB - не является клавишей для воспроизведения ноты.
Клавиша Ё(ё/~`) - отвечает за ноту ДО 3й октавы.
Q - нота РЕ третьей октавы.
U - нота РЕ четвертой октавы. U - PAGE UP.
PAGE UP - последняя белая клавиша четвертой октавы.
Черные ноты третьей октавы начинаются c клавиши 1. 12 456.
Черные ноты четвертой октавы начинаются c клавиши 8. 89 + BACKSPACE HOME.

++++++С помощью компьютерной мыши++++++

Нажимайте левой кнопкой мыши или колесиком на клавиши

=====Фичи=====

SOLO - Устанавливает состояние соло для этого канала.
Если соло верно, то будет звучать только этот канал и другие сольные каналы.
MONO - Включает или выключает монофонический режим.
В монофоническом режиме канал синтезирует только одну ноту за раз.
В режиме poly (идентично режиму mono off) канал может одновременно синтезировать несколько нот.
По умолчанию моно выключено (поли режим включен).
MUTE - Устанавливает состояние отключения звука для этого канала.
Значение true означает, что канал должен быть отключен, false означает,
что канал может звучать (если другие каналы не являются сольными)
VELOCITY - определяет громкость ноты.
Pressure - давление на клавиатуру.
Bend - смещение высоты тона вверх или вниз для этого канала.
Обратите внимание! Не все каналы поддерживают BEND/PRESSURE.
Reset Setting - ставит настройки Velocity,Bend,Pressure
в инзначальное состояние для этого канала

Панель инструментов - их 127:) Попробуйте, вещь прикольная.

=====Повороты=====

Было сложно куда-то вставить keyPressed для поворотов,
потому что остались почти только служебные клавиши.
Поэтому заранее извиняюсь за неудобное расположение этих клавиш.
Minus/-/_/Минус(клавиша рядом с 9/(/) отвечает за поворот вокруг оси OX+
0/)/Zero/Ноль/-/_ отвечает за поворот вокруг оси OX-
CONTEXT_MENU(клавиша рядом с alt gr/список) отвечает за поворот вокруг оси OY-
alt gr/alt(клавиша рядом с пробелом) отвечают за поворот вокруг оси OY+

=====Не корректное повидение=====

Если понажимать на Enter, то может пропасть звук
---решение понажимать еще раз и может быть заработает,или перезагрузить программу.
---это тоже системная клавиша, поэтому такое повидение возможно.
---в данной версии этой программы - эта проблема не пофикчена.
Когда вы выбирате инструмент на панели инструментов, то возможно
будет отсутствовать нажатие клавиши HOME и PAGE UP/PAGE DOWN
---Не волнуйтесь, это не баг.Так клавиши HOME и PAGE UP/PAGE DOWN являются системными
---Просто кликните мышью по другой кнопке(чтобы таб ушел с панели инструментов)
--- и все должно заработать снова.
Бывает, что не всегда нажиматся сразу несколько идущих последовательно клавиш,
---обычно на расстоянии все ок.
---Если не устраивать что-то мощное на этой клаве(поберегите свою клаву на ноуте),
---то все должно откликаться нормально
